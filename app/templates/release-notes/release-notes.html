Codebase

  • White-list and Black-List Framework in Place

    • The DMC has created a framework to be able to white-list and black-list endpoints. In the past these endpoints were listed one by one, but now we will be able to white-list and black-list endpoints via a singular file.

  • Quarantine Bucket Workflow

    • We have made changes to our document upload process to make it even safer. In the past, our infrastructure would scan all documents being uploaded to our site before making the document available for user to download. Documents that did not pass the virus and malware scan process would be marked as not safe and would not become available for download through our site. The change we have implemented will now move files marked as not safe into a separate storage solution, so we can more deeply analyze the threats our site may be vulnerable to, ensuring users are always safe when using our platform.

  • DMC License files have been updated

    • The DMC has updated all repositories stored on GitHub with new License Files. These Files can be found in the root of all repositories. The GitHub link can be found at: https://github.com/dmc-uilabs

Digital Manufacturing Commons Release V 0.1.0

The V.0.1.0 release of the Digital Manufacturing Commons platform was released on November 29, 2017. This release was marked as the alpha launch of the platform and was announced in a news letter sent out via UI LABS and was additionally announce at the Defense Manufacturing Conference in Colorado the same day. The major features included in this release include the DMDII Member Portal, DMDII Projects, User Profiles, My Organization, Individuals, Activities, My Workspaces, and All Workspaces. To get the features included with each of the major features above, please read below. The platform can be accessed at https://portal.opendmc.org/#/.
Read more about the DMC Alpha Release on our DMC wiki: https://digitalmfgcommons.atlassian.net/wiki/pages/viewpage.action?pageId=81035266
