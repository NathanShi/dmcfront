Introduction

During this release, the DMC heaviliy focused on making updates that effected the DMDII Membership Portal. The Membership Portal recieved a new look and feel (navigation hasn't changed too much so don't worry if you were a fan of how you traversed the portal before). In addition to these changes, we are slowly making the shift to releasing new content with every release we put out. We are starting this release strong with all the new static content pages we've added to the site which include our Security page and updated About and Activities pages.

We've also made a few updates to our legal page so definitely look over this when you have the opportunity.

As with most releases, we have added three more application to the DMC Marketplace in the form of external application contributions from DMDII Member Organizations such as Integris Engineering and Pennsylvania State University. Read more about them in our Applications section found below.

Finally, if you couldn't tell already, all release notes about the platform and the website will be published to this page going forward, so you no longer need to leave the DMC to learn more about the hard work that is put into each release as well as what new features and content is available to you every two weeks. All previous release notes can still be accessed from our wiki page on Confluence, with links to each article available in the tabs above.

Codebase

  • White-list and Black-List Framework

    • The DMC has created a framework to be able to white-list and black-list endpoints. In the past these endpoints were listed one by one, but now we will be able to white-list and black-list endpoints via a singular file. In future updates we will be utilizing this functionality to add full lists of white and blacklist on our own servers.

  • Quarantine Bucket Workflow Framework

    • We have made changes to our framework for our document upload process to make it even safer. In the past, our infrastructure would scan all documents being uploaded to our site before making the document available for user to download. Documents that did not pass the virus and malware scan process would be marked as not safe and would not become available for download through our site. When implimented, this will move files marked as not safe by our infrastructure will be moved into our quarentine storage solution. We will at this point be able to analyze in depth the threats our site experiences on a regular bases, to ensure our users are always safe when using our platform. This functionality will added in a future update while we are going through the process of testing this fuctionality as best we can befor adding it to our production platform.

  • DMC License files have been updated

    • The DMC has updated all repositories stored on GitHub with new License Files. These Files can be found in the root of all repositories. The GitHub link can be found at: https://github.com/dmc-uilabs

Applications

We've include three new application in our Marketplace for our user to demo. This time the application have stepped it up a notch. Two DMDII member organization have created applications that for a limited time, our current users will be able to try using for free. Try them while they are available.

These Applicaitions have been created by Integris Engineering and Pennsylvania State University. Learn more about that apps when visting the DMC Marketplace or at their associated Confluence Pages:

  • Single Shimmed Shear Join Calculator: The Single Shear Shimmed Joint app calculates an approximate minimum margin of safety (similar to a Safety Factor) on a bolted joint against a defined shear and tensile load.
    • App documentation can be found at: Single Shear Shimmed Joint by INTEGRIS
  • Conceptual Cost Model (Pennsylvania State University): Estimates the cost of a part or assembly.
    • App documentation can be found at: Conceptual Cost Model by Pennsylvania State University
  • Weld Cost Model (Pennsylvania State University): Estimate cost of welds.
    • App documentation can be found at: Weld Cost Model by Pennsylvania State University

Introduction

With the success of the Alpha release just a little bit over a month ago we are introducing our Beta release of the Digital Manufacturing Commons platform. Users will find a few new updates which include the new Marketplace tab, access to run a set of demo applications as well as the ability to see the full organizations list of organizations signed up to use the site. We've also introduced our live chatting tool to better help users use our site more effectively and efficiently.

During our Beta Launch period, only fully verified users on the website will be able to access the platform features, so if you are not verified or your organization administrator has not yet joined and verified their account, this is really the time to do so if you want to get a feel for what is to come in our full launch in the second quarter of 2017. To learn more about each of the Individual features, look in the use cases section, and as usual, if you have any questions please feel free to reach out. Modes of contact are all listed in the Contact Us section.

Special Friends of the DMC (Non DMDII Members) can sign up as well. During the sign up process, do not check the I am a DMDII Member checkbox. When done with the onboarding process, make sure to e-mail askdmc@uilabs.org and we will work to amke sure your account become verified.

Read more about the DMC Beta Release on our DMC wiki: V 0.1.1 Beta on Confluence.

Introduction

The V.0.1.0 release of the Digital Manufacturing Commons platform was released on November 29, 2017. This release was marked as the alpha launch of the platform and was announced in a news letter sent out via UI LABS and was additionally announce at the Defense Manufacturing Conference in Colorado the same day.

The major features included in this release include the DMDII Member Portal, DMDII Projects, User Profiles, My Organization, Individuals, Activities, My Workspaces, and All Workspaces. To get the features included with each of the major features above, please read below. The platform can be accessed at https://portal.opendmc.org/#/.
